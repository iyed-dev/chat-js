# chat-js

Pour installer "socket.io" 
npm install socket.io

ensuite écrivez dans le cmd

cd /répertoire/du/fichier


et pour installer "express" 
npm install express

ensuite écrivez dans le cmd

cd /répertoire/du/fichier

et écrivez ensuite 

node index.js

ensuite allez à l'adresse si-dessous 

http://localhost:3000/
